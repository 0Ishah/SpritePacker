This program is designed to create sprite sheets for animation using separate files or a gif animation.
To create a sprite sheet from the separate frames, first select the frames in the right order.
You can always add more frames or clear selected frames if needed. Keep in mind: image sizes must be
the same size for the animation to work properly. If the frames have different sizes, the final
sprite sheet will be resized.
To create a sprite sheet form a .gif animation, simply select the file, and the program will do
everything.
